⚙️ Core Features

Designed for researchers and clinicians who need precision, automation, and clarity.

🔊 Frequency Sweep Engine

Scans a user-defined or auto-calculated frequency range to find the resonant peak.

🔄 Auto-Track (PLL)

Continuously adjusts the frequency to maintain resonance even as tissue properties change.

🧩 Hardware Abstraction

Pure serial communication (RS-232/USB) with no simulation fallback — designed for real hardware.

🛑 Emergency Stop

Instantly disables output at the press of a button for maximum safety.

📁 Logging & Export

All session data and patient reports are saved as CSV and TXT files in the /logs folder.

🌍 Open Source

🚀 Quick Start

Get the software running in 3 minutes.

# 1. Clone the repository

git clone https://github.com/Khan-Ar/Anti-cancer--resonance--software.git

# 2. Install Python dependencies

pip install -r requirements.txt

# 3. Upload firmware to ESP32 (Arduino IDE)

Open firmware/resonance_controller.ino → Upload

# 4. Run the software

python main.py

Then go to the 📋 Treatment Planner tab, enter patient data, click Auto-Calculate, and hit Run Full Auto-Treatment.
