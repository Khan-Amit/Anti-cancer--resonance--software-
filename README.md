Command Response Description
SET_FREQ <Hz> OK Sets output frequency (e.g., SET_FREQ 1500000)
READ_PHASE -12.5 Returns phase in degrees (float).
READ_AMP 2.45 Returns amplitude in mV (float).
READ_FREQ 1500000 Returns current set frequency (optional).
